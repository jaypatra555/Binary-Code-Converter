How to Use The Converter

Step 1: Click the Converter button.

Step 2: Select the number system you want to convert from.

Step 3: Enter the number.

Step 4: Select the number system you want to convert to.

Step 5: Enter the number.

Step 6: The result will be displayed automatically.

How to Use The Calculator

Step 1: Click the Calculator button.

Step 2: Select the number system you calculate.

Step 3: Enter the number.

Step 4: The result will be displayed automatically.
